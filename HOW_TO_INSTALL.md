## Method 1: Load Extension in Chrome/Edge (Quick Test)

### Step 1: Open Extensions Page
1. Open **Chrome** or **Edge** browser
2. Type in address bar: `chrome://extensions/` (or `edge://extensions/`)
3. Press **Enter**

### Step 2: Enable Developer Mode
1. Look at the **top right corner**
2. Find the toggle switch that says **"Developer mode"**
3. Turn it **ON** (it will turn blue)

### Step 3: Load BrowserGenie Extension
1. Click the **"Load unpacked"** button (top left)
2. Navigate to this folder:
   ```
   D:\browser-warrior\BrowserGenie\packages\browsergenie-agent\dist
   ```
3. Click **"Select Folder"**

### Step 4: Use BrowserGenie! üéâ
- Extension will appear in your extensions list
- Click the **BrowserGenie icon** in toolbar
- Or press **Ctrl+E** (Windows) to open side panel
- Enjoy AI-powered browsing!

## Need Help?

If you face any issues:
1. Make sure Developer mode is ON
2. Check that you selected the correct `dist` folder
3. Refresh the extensions page
4. Try reloading the extension
